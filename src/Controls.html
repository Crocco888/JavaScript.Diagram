About this sample

This sample demonstrates some of the auxilliary controls available with JsDiagram: Palette, Overview and ZoomControl. The Palette represents a tool palette control that displays DiagramItem objects grouped into categories, which can be added to a diagram by drag-and-drop. The Overview provides a scaled-down view of the DiagramView content and lets the user scroll the bound view. The ZoomControl lets users zoom and pan the target DiagramView interactively.

This page requires a browser that supports HTML 5 Canvas element.
